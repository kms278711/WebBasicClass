Intellij 단축키 TOP15 표

단축키 설명 단축키 설명
Alt + Enter 코드에서 오류가 발견된 경우, 또한 경고 및 제안이 표시되는 곳에서 해당 오류 위에 커서를 놓고 단축키를 누르면 문제에 대한 수정 제안 목록이 표시된다. 아무 곳에서나 사용이 가능하며 적용 가느한 여러 액션이 표시된다. 자동 import시 편하게 사용가능하다. F2 제안 탐색 시 다음 오류, 경고, 제안으로 이동이 가능하다. 수정이 필요한 곳으로 이동하고 수정이 가능하다.
Cmd+1(Mac)
Alt+1(Windows) 		프로젝트 창을 열고 보고 있는 화면을 이동할 수 있다. 방향키를 사용해 트리를 탐색하고, 타이핑을 하여 검색할 수 있다. ESC 다른 파일 탐색 도중 에디터로 다시 창을 바꿀 수 있다. 변경 사항을 적용하지 않고 모든 팝업을 닫는데도 사용이 가능하다.
Cmd+E(Mac)
Ctrl+E(Windows) 		최근 열었던 파일을 확인가능하다. 단축키를 누르면 대화상자가 열려 방향키로 파일이 탐색 가능하다. 키보드 단축키가 없는 도구 창도 이용이 가능하다. Cmd+B(Mac)
Ctrl+B(Windows) 		코드 안에서 다른 코드로 이동하고 싶을 때 사용한다. 심볼의 선언으로 이동할 수 있다. 구현탐색 또한 가능하다.
Alt + F7 해당 코드가 사용된 모든 사용위치가 검색창에 표시된다. Ctrl 2번 IDE에서 어떤 위치에 있든, 혹은 어떤 파일이 열려 있든 'Run Anything'창이 열리면서 기본적으로 최근 실행한 실행 구성목록이 표시되며 검색 또한 가능하다.
⌥ + ↑, ⌥ + ↓(Mac)
Ctrl + W, Shift + Ctrl + W(Win/Linux) 		각각 앞에 있는 명령어가 확대, 뒤에 있는 명령어가 축소하는 단축키이다. 주변의 코드 섹션을 확장하거나 축소할 수 있다. 선택 영역 확장을 하면 섹션의 유효한 표현식을 자동으로 선택한다. Cmd+/(Mac)
Ctrl+/(Windows) 		간편하게 주석을 추가할 수 있다. 줄에서 사용할 경우 그 한줄이 주석이 되고 블록을 선택해서 할 경우 블록에 있는 코드가 주석 처리된다.
Shift + Cmd + Enter(Mac)
Shift + Ctrl + Enter(Windows) 		현재 선택한 구문을 완성시키는 기능이다. 세미콜론을 코드 끝에 추가하거나, if문 사용시 괄호 및 중괄호를 추가하고 커서가 올바른 위치에 놓이게 해준다. ⌥ + Cmd + L
Ctrl + Alt + L(Windows) 		현재 파일의 서식을 프로젝트 표준에 맞춰준다. 예를 들어 표준으로 맞춰야 하는 경우 표준으로 맞춰준다.
각각 ⌥ + Shift + Cmd + L, Shift+ Ctrl + Alt + L키로 서식을 변경 가능하다.
^ + T(Mac)
Shift + Ctrl + Alt + T(Win/Linux) 		심볼이나 선택 영역에서 적용 가능한 모든 리팩토링 옵션을 표시한다. Shift + Cmd + A
Shift + Ctrl + A(Windows) 		Intellij의 모든 액션을 검색할 수 있다. 액션 뿐만 아니라 단축키까지 나와서 연습할 수도 있다. 도구 창 검색 및 열기도 지원된다.
Shift 2번 전체 검색 단축키이다. 설정을 바꿀 수도 있고 검색어 입력 시 클래스, 파일, 심볼 및 액션의 검색 결과가 모두 표시된다.
